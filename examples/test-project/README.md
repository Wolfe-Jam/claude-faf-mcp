# Acme Dashboard

Internal analytics dashboard for Acme Corp. Built with Next.js 16, React 19, and Claude AI.

## Who

Engineering team at Acme Corp — 12 developers shipping internal tools.

## What

Real-time analytics dashboard showing customer usage patterns, revenue metrics, and AI-powered anomaly detection.

## Why

The old Grafana setup couldn't handle our custom metric types and the team needed AI-assisted anomaly alerts without context-switching to a separate tool.

## Stack

- **Frontend:** Next.js 16 + React 19
- **AI:** Claude API via Anthropic SDK
- **Database:** Postgres via Drizzle ORM
- **Hosting:** Vercel
- **Testing:** Vitest

## Getting Started

```bash
npm install
npm run dev
```
